# Genshin Impact Collection
- Français: <br>
Si vous souhaitez utiliser ce projet, vous trouverez un fichier `Query.sql` dans le même répertoire que ce fichier (oui, celui que vous lisez actuellement :P).
Vous pouvez l'utiliser pour créer la base de données et les tables requises pour que cela fonctionne, ainsi que certaines valeurs que vous pouvez insérer dans les tables (non obligatoire, vous pouvez créer vos propres éléments, unitclasses, origines et personnages si vous le souhaitez).
WampServer est utilisé pour ce projet. Assurez-vous qu'il est installé avec mySQL sur votre ordinateur avant de lancer le projet.
Enfin, la modification du fichier Config/Config.php peut être nécessaire pour que ce projet fonctionne. Modifiez le Data Source Name 'dsn', Username 'user' ou password 'pass' en conséquence.

- English: <br>
If you want to use this project, you will find a `Query.sql` file in the same directory as this file (yes, the one you are currently reading :P).
You can use it to create the database and tables required for this to work, aswell as some values that you can insert into the tables (not required, you can create your own elements, unitclasses, orignins and characters if you want to).
WampServer is used for this project. Please make sure it is installed with mySQL on your computer before running the project.
lastly, changing the Config/Config.php file might be neccary for this project to work. change the Data Source Name 'dsn, Username 'user' or password 'pass' accordingly.
